# DiceMannor GitHub Pages Site

This folder contains a tiny static website for App Store Connect:

- `index.html`
- `privacy.html`
- `support.html`

Before publishing, replace every `REPLACE_WITH_YOUR_SUPPORT_EMAIL` value in `privacy.html` and `support.html` with your real public support email address.

## Publish With GitHub Pages

1. Create a public GitHub repository, for example `dicemannor-site`.
2. Upload the files in this folder to the root of that repository.
3. In GitHub, open the repository's Settings tab.
4. Click Pages in the left sidebar.
5. Under Build and deployment, set Source to Deploy from a branch.
6. Set Branch to `main` and folder to `/root`.
7. Save.

Your URLs will look like:

```text
https://YOUR_GITHUB_USERNAME.github.io/dicemannor-site/privacy.html
https://YOUR_GITHUB_USERNAME.github.io/dicemannor-site/support.html
```

Use those two URLs in App Store Connect.
